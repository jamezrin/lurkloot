StreamMaxxer Privacy Policy

Last updated: May 31, 2026

StreamMaxxer does not collect, transmit, sell, or share user data.

Local Storage

The extension stores settings, scheduler state, campaign progress, managed tab identifiers, and diagnostic events locally in the user's browser using extension storage. This data remains on the user's device and is not sent to the developer or to third-party analytics services.

Platform Access

StreamMaxxer accesses Twitch and Kick pages only to provide its core drops farming functionality, including detecting campaigns, checking progress, managing visible muted tabs, and claiming eligible rewards. The extension uses the user's existing browser session and does not store, export, or transmit credentials, cookies, or authentication tokens.

Remote Code

The extension does not use remote code. All extension code is included in the installed extension package.

Contact

For privacy questions about StreamMaxxer, contact the developer through the project's GitHub repository.
